# FX01S

## 描述

- 10x11阵列的98%键盘，共100键
- 支持RGB轴灯
- Type-C接口
- 蓝牙主控为nRF52832
- 采用热拔插方式（凯华轴座）
- 支持旋转编码器
- 预留WS2812 RGB灯带焊接位
- OLED屏幕显示Caps Lock、Num Lock、Screen Lock指示功能
- PCB尺寸：390.6mm×119.1mm

## 指示灯说明
- OLED屏幕显示USB模式、蓝牙模式、充电指示、电量指示，键盘输入转态指示
- USB连接状态下，OLED屏幕常亮
- 蓝牙连接状态下，OLED屏幕5秒后自动熄灭（可自定义常亮时长）
- 蓝牙广播时闪烁30秒后自动熄灭。
