# GT Ω50 键盘 rev.C

## 描述

- 5x12阵列的50%键盘，共58键
- 支持RGB轴灯（QMK灯效）
- Type-C接口,引出外接USB接口
- 蓝牙主控为nRF52810（芯片+贴片天线）
- 采用热拔插方式（凯华轴座）
- 支持旋转编码器
- 轴灯兼指示灯功能
- 共引出6个针脚可自行添加外设
- PCB尺寸：PCB尺寸：247.7mm×95.3mm

## 指示灯说明
由第二排左起第一颗灯指示Capslock状态；由第一排左起第二颗灯指示键盘输出状态；

- 绿色-USB输出
- 青色-2.4G无线输出
- 蓝色-蓝牙通道1️⃣输出
- 红色-蓝牙通道2️⃣输出
- 橙色-蓝牙通道3️⃣输出
